HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The title is the only visible section of the head element to the user
  • The metadata is stored within the head element
  • An element is the entire section where as the tag refers only to what appears inside the angle brackets

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A color can be represented by either the color name or the Hex #.
  • There are three types of CSS. Inline, Internal style sheet and external style sheet.
  • A * can be used to declare ALL elements on the page have the style properties.
  • The CSS style sheet is linked within the header section of the HTML file

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git checkout *branch-name* to switch to this branch
  • git pull origin *current branch name* makes sure you have the up to date current branch
  • git push origin *feature branch name* pushs up all changes
  • git add -A stages the changes
  • git commit -m "comment here" enters the comment for the staged change

JavaScript

  • The JavaScript file is linked within the footer of the HTML file
  • Right click on browser window and select "Inspect" or use Control+Shift+I to open the Chrome DevTools. If on macOS use Command+Option+I.
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
